
 # 🛒 Superstore Sales Dashboard | Power BI

  An interactive and visually driven Power BI dashboard designed to analyze Superstore sales performance across multiple     
  business dimensions including sales, profit, customers, products, shipping, and regional insights.

  The dashboard provides executives and business users with dynamic KPIs, trend analysis, customer segmentation, and product 
  performance tracking to support data-driven decision making.
